# 1. 推荐可用库
|**库名称**|**特点**|**适用场景**|**GitHub 关键词**|
|---|---|---|---|
|**NodeGraphQt**|**最推荐**。UI 现代、功能完整、文档齐全。支持自定义节点、属性面板、Undo/Redo。|类似 Dify、Unreal Engine 蓝图、Nuke 风格的专业工具。|`NodeGraphQt`|
|**Ryven**|侧重于“执行流”。自带了一套基于 Python 的运行时环境，支持实时数据流。|需要边拖拽边执行代码，类似 LabVIEW 的逻辑。|`ryven`|
|**pyqt-node-editor**|相对轻量，适合学习原理，功能不如 NodeGraphQt 强大。|简单的逻辑编排。|`pyqt-node-editor`|

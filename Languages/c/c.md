# 1. 字面量后缀

*整型数据*

| 后缀            | 含义      | 对应类型                     | 说明               |
| ------------- | ------- | ------------------------ | ---------------- |
| 无             | 默认      | `int`                    | 编译器选择能容纳的最小有符号整型 |
| `u` / `U`     | 无符号     | `unsigned int`           | 常用于位运算、寄存器       |
| `l` / `L`     | 长整型     | `long int`               | 位宽与平台相关          |
| `ll` / `LL`   | 长长整型    | `long long int`          | C99 引入，至少 64 位   |
| `ul` / `UL`   | 无符号长整型  | `unsigned long int`      |                  |
| `ull` / `ULL` | 无符号长长整型 | `unsigned long long int` | 常用于地址、计数器        |
*浮点型*

|后缀|含义|对应类型|说明|
|---|---|---|---|
|无|默认|`double`|最常见、也是最容易踩坑的|
|`f` / `F`|单精度|`float`|MCU / DSP / 性能敏感代码必写|
|`l` / `L`|扩展精度|`long double`|精度与实现相关|

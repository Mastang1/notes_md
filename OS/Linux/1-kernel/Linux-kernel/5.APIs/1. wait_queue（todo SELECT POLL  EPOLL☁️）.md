## 1. 认知锚点 功能类比 事件标志组/事件list
rtos中的时间等待，是把TCB直接挂载到 wait_event list上，但是Linux的wait_queue是malloc一个结构体对象 entry，包含当前task的指针，插入到wait_queue中；_**简言之：**_ 一个是挂载TCB实体，一个是挂载TCB的指针；
_**目的：一个task可以等待多个事件。也就是把指向自己的指针插入到多个wait_queue中；当所有事件都没有到来，block当前task；当任意一个事件到来，就会唤醒task，task发起事件遍历，执行对应事件的user code，然后继续挂起task，等待下一个事件；**_

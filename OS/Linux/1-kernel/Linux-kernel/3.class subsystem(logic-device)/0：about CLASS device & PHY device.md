## 归纳
 - 1. 物理设备 和  逻辑设备概念。物理设备就是定义 或者内核根据dts生成的静态的device结构对象，包含硬件信息；逻辑设备概念，（以cdev为例），在进行device_create时候进行malloc的一个对象，包含 设备号等信息；
 - 2. （统一结构体的必要性）都用device这个相同的结构，为了在所有的代码中都可以以相同的函数对其操作，不必搞多套；
 - 3. 逻辑设备中可以通过parent指针获取物理device实体，（device_create时候传入）；
 - 4. 物理设备中可以通过私有child链表获取device实体，（device_create时候传入）；

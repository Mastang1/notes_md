## Initial - mmu
 - 1. 大概理解CPU把地址访问给到MMU，MMU通过查找页表进行物理地址访问；
 - 2. 访问中校验地址范围是否错误，常见的是缺页异常，进入该异常，分配物理内存映射；


## Initial - process running
 - 1. 应用程序进程，以文件形式test.elf存在磁盘上，有VFS进行管理；
 - 2. 进程创建后，所有的已初始化的data，都建立了VM - VFS的映射，也就是每个变量的具体初始信息存储在文件；
 - 3. 执行正常的CPU指令处理，因为没有分配页表，所以触发  缺页异常，进入异常处理；
 - 4. _**该异常处理中，执行了关联文件的查找，读取文件为对应变量分配了初始值**_
